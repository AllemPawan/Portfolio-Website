FastAPI Summarization API

A production-grade REST API for long document summarization with adjustable length, bullet points, key points extraction, streaming, and authentication.

Python FastAPI LLM Docker
View on GitHub

Overview

The FastAPI Summarization API is a production-ready service that transforms long documents into concise, structured summaries. It supports multiple output formats — paragraph, bullet points, and key points — with adjustable length controls.

Built with FastAPI, it features automatic Swagger documentation, API key authentication, a streaming endpoint for real-time generation, and a comprehensive test suite. Deploy anywhere with Docker.

Features

🔌

REST API

Well-documented RESTful endpoints for document summarization.

📖

Swagger UI

Interactive API docs at /docs for testing and integration.

📝

Long Document Summarization

Handles documents beyond LLM context windows via chunking and map-reduce.

📏

Adjustable Summary Length

Control output length with min/max words or sentences parameters.

📋

Bullet Summary

Generate concise bullet-point summaries for quick scanning.

🎯

Key Points

Extract the most important takeaways from any document.

⚡

Streaming Endpoint

Real-time streaming endpoint for token-by-token summarization.

🔑

Authentication

API key authentication to secure your summarization endpoint.

🐳

Docker Deployment

Optimized Docker image with health checks and production config.

✅

Unit Tests

Comprehensive test suite with pytest covering all endpoints and edge cases.

API Endpoints

POST /api/v1/summarize
  # Standard summarization
  { "text": "...", "style": "paragraph", "max_words": 200 }

POST /api/v1/summarize/bullet
  # Bullet-point summary
  { "text": "...", "max_bullets": 10 }

POST /api/v1/summarize/key-points
  # Key points extraction
  { "text": "...", "num_points": 5 }

POST /api/v1/summarize/stream
  # Streaming response (SSE)
  { "text": "...", "style": "paragraph" }

# All endpoints require: Authorization: Bearer <api_key>

Getting Started

# Clone & run
git clone https://github.com/AllemPawan/fastapi-summarization.git
cd fastapi-summarization
docker compose up -d

# API docs at http://localhost:8000/docs

# Quick test
curl -X POST http://localhost:8000/api/v1/summarize \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $API_KEY" \
  -d '{"text": "Long document text...", "style": "bullet"}'

# Run tests
docker compose run --rm app pytest
