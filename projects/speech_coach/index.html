Overview

Speech Coach is a live roleplay system for practicing spoken English. You speak naturally into your microphone, an AI conversation partner responds out loud via neural text-to-speech, and at the end of each session you receive a structured coaching report covering grammar corrections, fluency metrics, pronunciation trends, and prioritized improvement areas.

The entire speech pipeline (STT, pronunciation scoring, TTS) runs locally using open-source models — no cloud dependency, no API keys, no usage costs. The LLM for roleplay and judging is switchable between Claude API (best quality) and any local OpenAI-compatible server (Ollama, LM Studio, vLLM) for a fully offline setup.

Features

🎙️

Live Spoken Conversation

AI partner hears you via STT and speaks back via neural TTS — natural back-and-forth.

🎭

Roleplay Scenarios

Job Interview, Small Talk, Friendly Debate — each with distinct character prompts.

📊

Pronunciation Scoring

Phoneme-level accuracy via wav2vec2 — locally computed, no cloud API.

📝

Coaching Report

Structured feedback: strengths, grammar fixes, fluency notes, top 3 priorities, resources.

💻

Fully Local Option

Entire pipeline runs on your machine with Ollama + open-source models — zero cloud costs.

📈

Progress Tracking

Every session saved to JSON — track phoneme accuracy and fluency across sessions.

Architecture

flowchart TD USER["You (Microphone)"] --> STT["faster-whisper
Speech-to-Text"] STT --> LLM["LLM
Roleplay Partner"] LLM --> TTS["Piper TTS
Neural Text-to-Speech"] TTS --> USER STT --> PRON["wav2vec2
Pronunciation Scoring"] STT --> FLUENCY["Fluency Metrics
WPM, Filler Words"] PRON --> REPORT["Coaching Report
LLM + Aggregated Signals"] FLUENCY --> REPORT LLM --> REPORT REPORT --> LOG["session_logs/*.json"]

Getting Started

# Clone the repo
git clone https://github.com/AllemPawan/Portfolio-Website.git
cd Portfolio-Website/projects/speech_coach

# Install Python dependencies
pip install faster-whisper transformers torch phonemizer python-Levenshtein \
            sounddevice soundfile numpy anthropic openai piper-tts

# Install system dep for phonemizer (Linux)
sudo apt-get install espeak-ng

# Download a Piper voice
python3 -m piper.download_voices en_US-lessac-medium

# Set LLM provider (Claude API)
export LLM_PROVIDER="anthropic"
export ANTHROPIC_API_KEY="sk-ant-..."

# Run!
python main.py
