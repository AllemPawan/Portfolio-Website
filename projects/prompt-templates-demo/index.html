Prompt Templates Demo

A prompt engineering playground for creating, versioning, comparing, and testing prompt templates with live LLM output and full parameter control.

Python FastAPI LangChain Ollama Docker
View on GitHub

Overview

Prompt Templates Demo is an interactive tool for prompt engineering and experimentation. It provides a library of reusable prompt templates with variable substitution, versioning, and live comparison side-by-side.

Test different prompts, system instructions, and temperature settings against local LLMs. Save your best templates, export them for use in applications, and iterate on prompt design with real-time streaming output.

Features

📚

Prompt Template Library

Curated library of reusable templates for common tasks: summarization, analysis, coding, writing, and more.

🔤

Variable Substitution

Templates with {variable} placeholders for dynamic content insertion.

🔄

Prompt Versioning

Track changes and revert to previous versions of your prompts.

⚖️

Compare Multiple Prompts

Side-by-side comparison of different prompts with the same input to see which performs best.

💾

Save Templates

Persist your custom templates to the library for reuse.

📤

Export Prompts

Export templates as JSON or LangChain-compatible format for use in applications.

🌡️

Temperature Controls

Adjust creativity/randomness per test with real-time slider controls.

⚙️

System Prompt Editor

Dedicated editor for system prompts with character count and history.

⚡

Streaming Output

Real-time streaming of LLM responses as they are generated.

🐳

Docker Deployment

Containerized for consistent development and deployment.
