PDF Chat (RAG)

View on GitHub

Overview

PDF Chat (RAG) is a Retrieval-Augmented Generation system that lets you have natural conversations with your PDF documents. Upload one or more PDFs, and the system extracts text, chunks it intelligently, generates embeddings, and stores them in a FAISS vector index for fast semantic retrieval.

When you ask a question, the most relevant chunks are retrieved, scored by similarity, and passed to an LLM to generate a grounded answer with citations. The entire pipeline — ingestion, retrieval, generation — runs locally with full Docker support.

Features

📄

Upload Multiple PDFs

Batch-upload PDF documents for simultaneous processing and indexing.

📝

Extract Text

High-fidelity text extraction from PDFs using PyMuPDF and OCR fallback.

🧩

Chunk Documents

Intelligent text chunking with configurable size, overlap, and sentence boundaries.

🧠

Generate Embeddings

Create dense vector embeddings from chunks using local embedding models.

📊

Store in FAISS

Efficient vector indexing with FAISS for fast similarity search at scale.

🔍

Semantic Search

Find relevant passages using cosine similarity on dense embeddings.

💬

Ask Questions

Natural language Q&A over your documents with cited, grounded answers.

📋

Display Retrieved Chunks

View source chunks with similarity scores alongside each answer.

📈

Show Similarity Score

Confidence scores for retrieved chunks so you can assess relevance.

⚡

Streaming Responses

Token-by-token streaming for real-time answer generation.

💾

Save Vector Database

Persist FAISS indexes and metadata for reuse across sessions.

🐳

Docker Support

Multi-service Docker Compose setup for one-command deployment.
