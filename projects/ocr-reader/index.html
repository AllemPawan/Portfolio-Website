OCR Reader

Extract text from images with confidence scoring, region highlighting, translation, and batch processing via REST API.

Python EasyOCR Tesseract FastAPI Docker
View on GitHub

Overview

The OCR Reader is a production-grade Optical Character Recognition system that supports both EasyOCR and Tesseract backends. Upload images and get high-quality text extraction with per-character confidence scores and visual region highlighting.

Built as a REST API with FastAPI, it supports single image processing, batch OCR, text translation, and straightforward Docker deployment. Perfect for document digitization workflows.

Features

🖼️

Upload Images

Upload JPG, PNG, TIFF images via web UI or REST API.

📝

Extract Text

Accurate text extraction using EasyOCR or Tesseract backends.

📊

Confidence Score

Per-character and per-word confidence scores for quality assessment.

🔦

Highlight Detected Regions

Visual overlay showing detected text regions on the original image.

🌐

Translate Extracted Text

Translate OCR results into multiple languages via LLM integration.

📥

Download Extracted Text

Export results as TXT, JSON, or structured Markdown.

📦

Batch OCR

Process multiple images in one request with parallel processing.

🔌

REST API

Full REST API with Swagger docs for programmatic integration.

🐳

Docker Deployment

GPU-enabled Docker image with both EasyOCR and Tesseract.

Getting Started

# Clone & run
git clone https://github.com/AllemPawan/ocr-reader.git
cd ocr-reader
docker compose up -d

# Open http://localhost:8000
# Upload images and get instant OCR results

# Or use the REST API
curl -X POST http://localhost:8000/api/ocr \
  -F "image=@document.jpg" \
  -F "engine=easyocr"
