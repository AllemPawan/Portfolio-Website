SQL AI Assistant

View on GitHub

Overview

The SQL AI Assistant lets anyone query a SQLite database using natural language. Upload a database, and the AI reads the schema, generates safe SELECT queries, executes them, and explains the results in plain English with auto-generated charts.

Built with FastAPI and Ollama, it prioritizes SQL safety — only SELECT queries are allowed, and the LLM is constrained to use only real tables and columns from the schema.

Features

📤

Database Upload

Upload any SQLite database. Schema is read automatically.

💬

Natural Language Queries

Ask questions in plain English — "Show top 10 customers", "Average sales by month".

🛡️

SQL Safety

Only SELECT queries allowed. DELETE, DROP, UPDATE, INSERT, ALTER, CREATE are rejected.

📊

Auto Charts

Bar, line, and pie charts auto-generated from numeric columns using Chart.js.

📥

Export Results

Export query results as CSV, Excel, or SQL.

🧠

AI Explanation

After execution, the AI explains what the query did and the business meaning of results.

💾

Chat History

Conversation memory — previous questions influence future queries.

🐳

Docker Support

One-command deployment with Docker Compose.

API Endpoints

POST /api/upload-db    Upload SQLite database file
GET  /api/schema       Get database schema (tables, columns, types)
GET  /api/tables       List table names and row counts
POST /api/query        Ask a question — returns SQL, results, explanation
GET  /api/history      Get conversation history
GET  /api/export/:fmt  Export results as csv / excel / sql
GET  /api/health       Health check

Getting Started

# Clone & run
git clone https://github.com/AllemPawan/sql-ai-assistant.git
cd sql-ai-assistant

# Configure environment
cp .env.example .env
# Edit .env with your Ollama endpoint

# Run with Docker
docker compose up -d

# Open http://localhost:8000
# Upload a .db file and start asking questions!
