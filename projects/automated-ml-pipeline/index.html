Features

📊

Auto EDA

Comprehensive exploratory data analysis — statistics, distributions, correlation heatmaps, missing value analysis, and class balance visualization.

⚙️

Auto Preprocessing

Smart feature type detection, missing value imputation, categorical encoding, and feature scaling — no manual configuration needed.

🧠

Multi-Model Training

6 classification and 7 regression algorithms with automatic problem type detection. Compare Logistic Regression, Random Forest, XGBoost, SVM, and more.

🎯

Hyperparameter Tuning

GridSearchCV-based hyperparameter optimization with sensible parameter grids for each algorithm. Toggle on/off as needed.

🏆

Model Comparison

Side-by-side metrics comparison, confusion matrices, and visual charts to identify the best performing model for your data.

📦

Model Export

Download trained models as .pkl files ready for production deployment. Session management keeps your work organized.

How It Works

1

Upload CSV

Drag and drop any CSV file. The pipeline immediately runs a full EDA and presents statistics, visualizations, and data quality metrics.

2

Configure Training

Select the target column, choose algorithms, set test split ratio, and optionally enable hyperparameter tuning. Problem type (classification/regression) is auto-detected.

3

Train & Compare

All selected algorithms train in parallel. View detailed metrics, confusion matrices, and comparison charts to pick the best model.

4

Export & Deploy

Download any trained model as a serialized .pkl file. Sessions persist on the server for later review.
